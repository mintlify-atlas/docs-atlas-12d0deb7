## Prerequisites

Before you begin, make sure you have the following:

- A **Firebase project** with Authentication enabled
- A PayPac user account with the `ORGANIZER` or higher role

<Note>
  The PayPac backend does not handle login directly. Firebase manages authentication on the client side. Your app obtains a Firebase ID token and passes it to the API.
</Note>

## Make your first API call

<Steps>
  <Step title="Authenticate via Firebase to get an ID token">
    Sign in using the Firebase client SDK in your app. After a successful sign-in, retrieve the ID token:

    <CodeGroup>

    ```javascript JavaScript (Firebase SDK)
    import { getAuth, signInWithEmailAndPassword } from "firebase/auth";

    const auth = getAuth();
    const userCredential = await signInWithEmailAndPassword(auth, "user@example.com", "your-password");
    const idToken = await userCredential.user.getIdToken();
    ```

    ```typescript TypeScript (Firebase SDK)
    import { getAuth, signInWithEmailAndPassword } from "firebase/auth";

    const auth = getAuth();
    const userCredential = await signInWithEmailAndPassword(auth, "user@example.com", "your-password");
    const idToken: string = await userCredential.user.getIdToken();
    ```

    </CodeGroup>

    <Tip>
      Firebase ID tokens expire after 1 hour. Call `user.getIdToken(true)` to force a refresh before making API requests if the token may be stale.
    </Tip>
  </Step>

  <Step title="Verify your identity">
    Use the token as a `Bearer` credential in the `Authorization` header. The `GET /api/auth/me` endpoint returns the profile of the currently authenticated user.

    <CodeGroup>

    ```bash cURL
    curl --request GET \
      --url https://api.paypac.com.co/api/auth/me \
      --header "Authorization: Bearer <firebase_id_token>"
    ```

    ```javascript JavaScript (fetch)
    const response = await fetch("https://api.paypac.com.co/api/auth/me", {
      method: "GET",
      headers: {
        Authorization: `Bearer ${idToken}`,
      },
    });

    const profile = await response.json();
    console.log(profile);
    ```

    </CodeGroup>
  </Step>

  <Step title="Create your first event">
    Once authenticated, you can create an event using `POST /api/events`. This endpoint requires the `ORGANIZER` role or higher.

    <CodeGroup>

    ```bash cURL
    curl --request POST \
      --url https://api.paypac.com.co/api/events \
      --header "Authorization: Bearer <firebase_id_token>" \
      --header "Content-Type: application/json" \
      --data '{
        "name": "Summer Music Festival",
        "short_description": "An outdoor music festival with live performances.",
        "description": "Join us for a full night of live music across three stages in the heart of the city.",
        "date_event": "2026-07-15T18:00:00.000Z",
        "place_address": "Parque Simón Bolívar, Bogotá",
        "city": "Bogotá",
        "country": "Colombia",
        "currency": "COP",
        "event_type": "PUBLICO",
        "type_venue": "SIN_NUMERAR"
      }'
    ```

    ```javascript JavaScript (fetch)
    const response = await fetch("https://api.paypac.com.co/api/events", {
      method: "POST",
      headers: {
        Authorization: `Bearer ${idToken}`,
        "Content-Type": "application/json",
      },
      body: JSON.stringify({
        name: "Summer Music Festival",
        short_description: "An outdoor music festival with live performances.",
        description: "Join us for a full night of live music across three stages in the heart of the city.",
        date_event: "2026-07-15T18:00:00.000Z",
        place_address: "Parque Simón Bolívar, Bogotá",
        city: "Bogotá",
        country: "Colombia",
        currency: "COP",
        event_type: "PUBLICO",
        type_venue: "SIN_NUMERAR",
      }),
    });

    const event = await response.json();
    console.log(event);
    ```

    </CodeGroup>
  </Step>

  <Step title="Check the response">
    A successful event creation returns `201 Created` with the new event object:

    ```json 201 Created
    {
      "message": "Evento creado exitosamente",
      "data": {
        "id": 42,
        "name": "Summer Music Festival",
        "short_description": "An outdoor music festival with live performances.",
        "date_event": "2026-07-15T18:00:00.000Z",
        "place_address": "Parque Simón Bolívar, Bogotá",
        "city": "Bogotá",
        "country": "Colombia",
        "status": "CREATED",
        "event_type": "PUBLICO",
        "type_venue": "SIN_NUMERAR"
      }
    }
    ```

    If your token is missing or expired, you will receive a `401 Unauthorized` response. If your role does not permit the action, you will receive `403 Forbidden`.

    <Warning>
      Events start in `CREATED` status and must be approved by a `PAYPAC` admin before they can go on sale.
    </Warning>
  </Step>
</Steps>
